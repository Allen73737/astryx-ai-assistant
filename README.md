## ⚙️ INSTALLATION & SETUP

Step into the future in just a few commands.

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Allen73737/astryx-ai-assistant.git
cd astryx-ai-assistant
```

### 2️⃣ Initialize the Frontend (Electron + React)
```bash
npm install
npm run dev
```

### 3️⃣ Initialize the Backend (Python + FastAPI)
```bash
cd backend
python -m venv venv
# On Windows:
venv\Scripts\activate
# On Mac/Linux:
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

Astryx leverages **Zustand** for blazing fast, globally synced frontend state, paired with a robust **Python Backend** utilizing WebSockets for zero-latency bidirectional streaming. When you speak, Astryx listens, processes, and acts across your entire operating system.

- **Trend Learner:** Constantly updates its knowledge graph based on your interactions.
